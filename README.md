# .dotfiles

## Requirements
* zsh
* git
* [zplug](https://github.com/zplug/zplug)
* [Awesome Patched Terminal Fonts](https://github.com/gabrielelana/awesome-terminal-fonts/tree/patching-strategy/patched)

## Installation
```
$ git clone https://github.com/mfakane/.dotfiles.git
$ cd .dotfiles
$ make
```
